Geodesic solutions from a metric with Julia

A recent interest of mine has been integrating geodesic equations using DifferentialEquations.jl, for which I've been developing a small package. One of the aims of the project is to provide a symbolic backend for generating the needed Julia functions, and automatically integrating to trace geodesic paths in arbitrary metrics.
